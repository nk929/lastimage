# Image Prompt Builder - 이미지 프롬프트 생성기

AI 이미지 생성을 위한 완벽한 프롬프트를 체계적으로 구성할 수 있는 정적 웹 애플리케이션입니다.

## 🎯 현재 구현된 기능

### ✅ 완료된 기능

#### 🏗️ 체계적 프롬프트 구성 시스템
- **7개 주요 카테고리**: 주제, 상세설명, 스타일, 카메라/구도, 화면 비율, 조명, 품질/디테일
- **카테고리별 기본 옵션**: 각 분야에서 자주 사용되는 옵션들을 버튼으로 제공
- **커스텀 입력**: 모든 카테고리에서 직접 입력 가능
- **실시간 프롬프트 생성**: 선택사항 변경 시 자동으로 프롬프트 업데이트

#### 🔄 양방향 프롬프트 처리
- **자동 합성**: 선택된 요소들을 영문/한글 프롬프트로 자동 조합
- **이중 언어 지원**: 영문과 한글 프롬프트 동시 생성
- **🆕 한국어 문법 구조 보존 시스템**: 한글 프롬프트 편집 시 연결된 동사/형용사의 어미가 불필요하게 재번역되지 않도록 보호
- **🆕 완전한 한→영 번역 보장**: 한글 프롬프트 직접 입력/수정 시 모든 한국어가 영어로 완전히 번역되어 영문 프롬프트에 한글이 절대 남지 않음
- **🆕 한국어 조사 및 문법 요소 처리**: '에서', '을', '를', '이', '가' 등 한국어 조사를 자동으로 제거하거나 적절한 영어 전치사로 변환
- **🆕 영문법 자동 개선**: 번역 후 부자연스러운 영어 표현을 자동으로 수정 ('doing is' → 'doing', 'pants in jeans' → 'jeans pants' 등)
- **🆕 실시간 전체 번역**: 한국어가 포함된 경우 무조건 전체 번역을 수행하여 누락 없는 완벽한 번역 보장
- **🆕 프롬프트 생성 시 완전 분리**: 옵션 선택/커스텀 입력 시 영문과 한글 프롬프트가 완전히 독립적으로 생성되어 언어 혼재 방지
- **🆕 선별적 번역 시스템**: 수정된 부분만 정확하게 번역하여 기존 한국어 문법 요소 보존
- **🆕 확장된 번역 사전**: 400+ 한국어 단어 지원으로 일반적인 한국어 표현의 완벽한 번역
- **🆕 고급 문장 번역**: 긴 한국어 문장도 완벽하게 영어로 번역하는 고도화된 번역 엔진
- **🆕 형태소 분석 유사 기능**: 사전에 없는 복합어도 부분 번역으로 처리
- **실시간 양방향 번역**: 영문 프롬프트 수정 시 한글 자동 번역, 한글 프롬프트 수정 시 영문 자동 번역
- **붙여넣기 자동 번역**: 프롬프트 붙여넣기 시 상대 언어로 즉시 번역
- **커스텀 입력 자동 번역**: 각 요소의 추가 입력칸에 한글 입력 시 영문으로, 영문 입력 시 한글로 자동 번역하여 프롬프트 생성
- **🆕 어근 기반 번역**: 한국어 완성형 문장에서 어근만 추출하여 번역, 문법적 활용형은 그대로 보존

#### 📋 편리한 관리 기능
- **다양한 복사 옵션**: 영문, 한글, 또는 전체 프롬프트 복사
- **개별 프롬프트 관리**: 각 프롬프트별 개별 복사/삭제 버튼
- **실시간 통계**: 선택된 옵션 수, 커스텀 입력 수, 단어 수 표시
- **전체 초기화**: 모든 선택사항과 입력 내용 한번에 초기화

### 📋 주요 진입점 및 기능

#### 메인 페이지 (`index.html`)
- **경로**: `/`
- **기능**: 이미지 프롬프트 생성의 모든 기능 통합

#### 핵심 카테고리별 기능

1. **주제 (Subject)**
   - 옵션: 사람, 동물, 풍경, 건축물, 사물, 차량, 음식, 추상적
   - 커스텀 입력: 구체적인 주제 직접 입력

2. **상세 설명 (Description)**
   - 옵션: 아름다운, 상세한, 사실적인, 우아한, 드라마틱한 등
   - 커스텀 입력: 추가적인 형용사나 설명

3. **스타일 (Style)**
   - 옵션: 포토리얼리스틱, 애니메이션, 유화, 수채화, 디지털 아트 등
   - 커스텀 입력: 특수 예술 스타일

4. **카메라 & 구도 (Camera & Composition)**
   - 옵션: 클로즈업, 미디엄샷, 와이드샷, 조감도, 저각도 등
   - 커스텀 입력: 전문적인 카메라 설정

5. **🆕 화면 비율 (Aspect Ratio)**
   - 옵션: 1:1 (정사각형), 4:3 (가로형), 16:9 (와이드스크린), 9:16 (세로형), 3:4 (표준), 21:9 (울트라와이드), 2.35:1 (시네마), 3:1 (배너)
   - 커스텀 입력: 특수 비율 설정 (예: 5:4, 7:5, 16:10)
   - AI 이미지 생성 서비스 형식: --ar 플래그 자동 생성

6. **조명 (Lighting)**
   - 옵션: 자연광, 골든아워, 블루아워, 스튜디오, 드라마틱 등
   - 커스텀 입력: 특수 조명 효과

7. **품질 & 디테일 (Quality & Details)**
   - 옵션: 매우 상세한, 걸작, 최고 품질, 8K, 선명한 초점 등
   - 커스텀 입력: 추가 품질 키워드

## 🎯 사용법

### 기본 사용 방법
1. **카테고리별 선택**: 원하는 옵션 버튼들을 클릭하여 선택
2. **커스텀 입력**: 추가적인 요소는 각 섹션의 입력칸에 직접 입력
3. **자동 생성**: 선택사항 변경 시 우측 프롬프트가 자동 업데이트
4. **복사 사용**: 생성된 프롬프트를 복사하여 AI 이미지 생성 도구에서 활용



### 정밀 선별적 번역 시스템
1. **차이 감지**: 이전 텍스트와 현재 텍스트를 비교하여 변경된 부분만 식별
2. **선별적 번역**: 수정된 단어/구문만 번역하고 나머지는 그대로 유지
3. **매핑 관계 저장**: 번역된 요소간 1:1 대응 관계를 기억하여 일관성 유지
4. **즉시 반응**: 텍스트 수정과 동시에 해당 부분만 즉시 번역
5. **기존 내용 보존**: 수정하지 않은 다른 단어들은 절대 변경되지 않음
6. **정확한 위치 매핑**: 추가/삭제/수정된 요소의 정확한 위치를 추적하여 반영

### 🆕 한국어 문법 구조 보존 시스템
#### 문제점 해결
- **기존 문제**: 한국어 프롬프트에서 단어 수정 시 연결된 동사/형용사의 어미까지 불필요하게 재번역
- **해결책**: 한국어 문법 요소 인식 및 보호 시스템 구현

#### 핵심 기능
1. **완성형 한국어 구문 인식**: 
   - 동사/형용사 활용형 패턴 감지 (`하다`, `이다`, `되다`, `있는`, `한`, `는` 등)
   - 조사 패턴 인식 (`이/가`, `을/를`, `에/에서`, `으로/로` 등)
   
2. **어근 기반 번역**:
   - 완성형 한국어 구문에서 어근만 추출하여 번역
   - 문법적 어미와 조사는 제거하여 영어로 번역
   - 예: `아름다운` → `beautiful` (어미 `-은` 제거)
   
3. **완전한 번역 보장**:
   - 한국어가 포함된 텍스트는 무조건 전체 번역 수행
   - 영문 프롬프트에 한국어가 절대 남지 않도록 보장
   - 실시간 번역으로 입력과 동시에 즉시 번역 반영
   
4. **기존 문법 요소 보호**:
   - 이미 한국어로 완성된 문법 구조는 번역 대상에서 제외
   - 새로 추가된 요소만 선별적으로 번역
   - 수정된 부분도 문법적 완성도에 따라 차별적 처리

#### 지원되는 한국어 패턴
```javascript
// 동사 어미
하다, 이다, 되다, 가다, 오다, 보다, 있다, 없다

// 관형형 어미  
한, 는, 은, 을, 된, 있는, 하는, 되는, 가는

// 조사
이/가 (주격), 을/를 (목적격), 에/에서 (부사격), 
으로/로 (부사격), 와/과 (부사격), 에게 (부사격)

// 어미 변형
로운, 운, 인 (형용사 활용)
```

### 커스텀 입력 스마트 번역
1. **각 요소 섹션의 추가 입력칸**: 주제, 설명, 스타일 등 모든 섹션의 커스텀 입력칸 활용
2. **언어별 자동 처리**:
   - **한글 입력**: 영문 프롬프트에는 영어로, 한글 프롬프트에는 그대로 표시
   - **영문 입력**: 영문 프롬프트에는 그대로, 한글 프롬프트에는 한국어로 표시
   - **혼합 입력**: 각 언어 부분을 해당 프롬프트에 맞게 번역
3. **즉시 반영**: 입력 후 "프롬프트 생성" 버튼 클릭 시 언어별로 적절히 번역된 내용으로 프롬프트 생성

### 개별 프롬프트 관리
1. **개별 복사**: 각 프롬프트 우상단의 복사 버튼으로 개별 복사
2. **개별 삭제**: 각 프롬프트 우상단의 삭제 버튼으로 개별 삭제
3. **전체 관리**: 하단 버튼들로 전체 복사/삭제 가능
